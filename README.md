# Windeck

Launcher a windows rendszert futtató Steam Deck konzolokhoz.

### Felhasznált keretrendszerek

- Tauri
- React

### Szkriptek

###### Tauri Build
`npm run tauri build`

###### Vite+Tauri dev server (Frontend & Backend teszteléséhez)
`npm run tauri dev`

###### Vite dev server (Frontend teszteléshez)
`npm run dev`
